以前Houdini只有單純的安裝檔(Houdini 18以前)，現在19版改成兩種方式，LAUNCHER與INSTALLER，而官方推薦第一種，安裝之後如圖一所示。

#接下來跟大家說明以上這些東東是什麼?(能吃嗎?)

**Houdini :**主程式(廢話)，又分為PRODUCTION BUILDS(大概每個月更新，較穩定的版本)、DAILY BUILDS(大約每天更新，主要為修復或新增功能)與Choose Another Version(選擇其他版本)。

Engine Plug-Ins: 是官方為了與其他3D軟體做連結而設計的外掛，可將數位資產Digital Assets(HDA)轉到其他3D軟體使用，早期有3Ds max、Maya、Cinema 4D、Unity、Unreal等，可付費使用，現在則取消支持Cinema 4D，開放Unity、Unreal免費使用。

**HQueue:**主要作為遠端或算圖農場，個人玩家超少用，有興趣可以去研究，這邊就不多作說明。

Labs/Packages: 必裝，以前好像叫Game Dev Toolset專為遊戲開發的工具，現在改名為SideFX Labs(實驗室工具)，裡面有很多好玩有用的節點，可以深入研究。

**Licenses:**Houdini管理許可證的地方，建議安裝最新版本的就可以，這裡有個要抱怨的地方，如果你是購買節點鎖定的版本(會登記電腦名稱)，記得在你換電腦安裝Houdini之前要記得!!!把許可證退出把許可證退出把許可證退出~，方法如下。

#如何退回許可證？

條件:

  • 使用 許可證管理員 (hkey) 或 sesictrl（非圖形 hkey）。
  • 返還許可證必須由計算機本身發起。
  • 許可證 不能 從網站退回 。
  • 許可證每年最多可退還 10 次。

返回許可證將從電腦名稱和硬體中釋放它們。它們將作為已刪除的權利放回您的 SideFX 許可帳戶，可以使用許可管理器從另一台電腦上提取這些權利。

當您回收許可證時，會計算禁用代碼並將其寫入電腦上的許可證文件。 禁用代碼也會被記錄並添加到您的帳戶中。許可證 ID 隨即失效。

#有兩種方法可以返回許可證

1.許可證管理員（hkey）：

  • 啟動許可證管理員。
  • 選擇許可證。
  • 右鍵單擊並選擇“返回許可證”。
  • 輸入您的電子郵件地址和用於許可的密碼。

從 Houdini 19.0 開始，現在支持部分返回。您可以退回全部數量的許可證子集。

2.非圖形命令行工具 sesictrl:

  • 該 sesictrl 程序管理許可證沒有圖形界面的方式。
  • 要訪問 sesictrl，您需要設置 Houdini 命令行環境 。

在終端中設置好 Houdini 環境後，使用 sesictrl 返回許可證：

  1. 輸入 sesictrl login 登錄到您的 SideFX 許可證帳戶。
  2. sesictrl print-license 獲取已安裝許可證的列表（第一列是許可證 ID）。
  3. sesictrl return-license LicenseID 啟動返回。許可證 ID 是要返回的許可證的數字 ID。

請注意，sesictrl 不支持部分返回。您必須使用許可證管理員 (hkey)。 要返回多個許可證，請提供需要返回的許可證的逗號分隔列表。

發表於 2022-01-16 更新於 2022-06-22 分類於 初學到入門

Houdini是由Side Effects Software Inc(加拿大公司)所開發，是基於節點模式(如圖一)的軟體，可運行於Linux、Windows、Mac OS等作業系統。

圖一

#目前Houdini分為五種版本:

HOUDINI APPRENTICE：學習版，只要註冊帳號即可申請使用，有不能商業使用，不支持Houdini Engine，輸出分辨率最高1280x720且帶水印等限制，副檔名為.hipnc、.hdanc。

HOUDINI INDIE：獨立遊戲版，有限的商業使用，可公司用但年收不超過10萬美元，副檔名為.hiplc、.hdalc。

HOUDINI ARTISTS：藝術家版，最多五個許可證使用(電腦會被鎖定)，功能上與FX完整版沒有差異，副檔名為.hip、.hda。

HOUDINI STUDIOS：工作室版，浮動的許可證，價格最高，副檔名為.hip、.had。

HOUDINI EDUCATION：教育版，專為學校和培訓中心所設計的完整版，需聯繫官方購買，副檔名為.hipnc、.hdanc。
